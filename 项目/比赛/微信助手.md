# 前后端接口

后端可以按照这个架构来：

* project

  * dao （数据库）

    * WeChatDao.py

      实现读取微信数据的功能

      * 用户信息
      * 聊天记录
  * service （前后端交互）

    * WeChatService.py

      * 调用dao的函数获取信息
  * view （前端）
